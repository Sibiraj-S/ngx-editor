The options provided directly will be considered as the primary option.

Property Type Default Accepted Values Description
placeholder string Enter text here... any The placeholder attribute specifies a short hint that describes the expected value of an editor field.
spellcheck boolean true boolean The spellcheck attribute specifies whether the editor is to have its spelling and grammar checked or not.
editable boolean true boolean The Editable attribute specifies whether the editor to be editable or not.
translate string yes yes/no The translate attribute specifies whether the content of an editor should be translated or not. This is an experimental technology and not supported by any browsers yet.
height string auto here The height property sets the height of the editor.
minHeight string 0rem here The minHeight property is used to set the minimum height of the editor.
width string auto here The width property sets the width of the editor.
minWidth string 0rem here The minWidth property is used to set the minimum width of the editor.
showToolbar boolran true boolean show / hide toolbar
toolbar array of arrays here here The toolbar property sets toolbar for the editor.
